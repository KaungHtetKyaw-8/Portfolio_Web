  • Frontend: Thymeleaf, HTML, CSS, JavaScript, Bootstrap 5

🔹 Main Features:

  • Employee Management: Create, update, and manage employee profiles and records
  • Customer Management: Register and manage customer data and point card integration
  • Inventory Management: Track product inventory and manage stock levels
  • Pos System : Streamlined product selection, order creation, and real-time inventory update

  • Security : Available in both English and Japanese

  • Database: MySQL

従業員、顧客、在庫、POS、注文など、主要な業務データを一元的に管理するためのフルスタックWebアプリケーションです。Java Spring Bootとモダンなフロントエンド技術を使用して構築され、実践的なCRUD操作、クリーンアーキテクチャ、レスポンシブなUI、セキュリティ対策を備えています。

🔹 使用技術:

  • バックエンド: Spring Boot, Spring MVC, Spring Web, Spring Jpa, Spring JDBC, Spring Security
  • フロントエンド: Thymeleaf, HTML, CSS, JavaScript, Bootstrap 5

🔹 主な機能:

  • 従業員管理： 従業員のプロフィールと勤務情報を管理
  • 顧客管理： 顧客情報とポイントカードの管理
  • 在庫管理： 商品の在庫や詳細情報を一元管理
  • POSシステム： 商品IDやカテゴリUIを用いて注文処理が可能。リアルタイムで在庫を反映
  • 注文管理： 顧客・商品情報と連携した注文情報の登録と管理
  • 多言語対応： 英語・日本語の表示に対応
  • レスポンシブデザイン： Bootstrap 5 によるスマートフォン対応のレイアウト
  • クリーンアーキテクチャ： SpringエコシステムによるMVCパターンに基づく構成

🔹 セキュリティ:

  • 事前共有キー認証 によるアクセス制限
  • IPアドレス制限による外部からの不正アクセス防止
  • 入力バリデーションとエラーメッセージの表示によるデータ整合性の確保

🔹 デプロイ情報:

  • ホスティング： AWS Lightsail（Ubuntu 24.04 LTS）
  • Webサーバー： Tomcat,Nginx
  • データベース： MySQL
  • SSL/HTTPS： CertbotとLet’s Encrypt によるHTTPS対応
  • リモートアクセス： SSHトンネル・ファイアウォール設定による安全な運用

本システムは、保守性が高くユーザーフレンドリーな業務アプリケーションをJavaとWeb技術で構築できるスキルを示すためのポートフォリオです。
注意： 本アプリケーションは就職活動向けに作成されたものであり、データはすべて架空です。
